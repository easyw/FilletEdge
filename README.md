# FilletEdge - KiCAD

FilletEdge is a KiCAD plugin to fillet the edge for pcb boards.

- Easy to fillet rect & lines
- Break Lines
- Split Rect to lines

## Know Issues

- Non 90 degree angle lines may throw error during the length check.
